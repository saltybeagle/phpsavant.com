allowedStatic (line 616)

A list of allowed static method calls for templates.

The format is ...

array( 'Class1' => array('method1', 'method2'), 'Class2' => array('methodA', 'methodB'), 'Class3' => '*' );

If you want to allow all methods from the static class to be allowed, use a '*' in the method name list.
